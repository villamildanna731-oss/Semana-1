# 🏛️ Proyecto: Papelería y Fotocopias

## 📌 Descripción

Este proyecto modela un sistema básico de papelería que incluye productos de oficina y servicios de impresión.

Permite gestionar pedidos de clientes, calcular totales y filtrar pedidos por estado.

## 🧱 Entidades

- Product: productos de papelería
- PrintService: servicios de impresión
- Customer: clientes
- Order: pedidos

## 🧠 Conceptos aplicados

- Interfaces de TypeScript
- Type unions (`Item`)
- Type literals (`OrderStatus`)
- Funciones tipadas

## ⚙️ Funcionalidades

- Crear productos y servicios
- Crear pedidos
- Calcular el total de pedidos
- Filtrar pedidos por estado
